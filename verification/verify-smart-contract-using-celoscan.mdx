---
title: "Verify Smart Contract using CeloScan"
sidebarTitle: "Using Celoscan"
description: "Verifying a smart contract allows developers to review your code from within the CeloScan Block Explorer"
---

- Navigate to the **Contract** tab at the Explorer page for your contract's address
- Click **Verify & Publish** to enter the smart contract verification page

![github](/images/image1-4acb0b29455cbf25fb8f78188f1104f0.png)

- Select the **Compile** type, **version** and **license**.

![github](/images/image2-73f87a6e64f34e915a6f3bba993b5f3c.png)

- Enter the Solidity Code of the contract along with the constructor arguments.
- Complete Captcha and click **Verify and Publish**.

![github](/images/image3-2c5ab226c92f27bc886c2ebd01e6091d.png)

- If done correctly, you should see the following screen.

![github](/images/image4-836d07669775f701757f89446734b97e.png)
